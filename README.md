## التثبيت 
# انسخ الاوامر بل سرة 

## 1
```bash
git clone --depth=1 https://github.com/sajjad-salam/theme-termux.git
```
## 2
```bash
cd theme-termux
```

## 3
```bash
export COLUMNS LINES
```

## 4
```bash
./install.sh
```
